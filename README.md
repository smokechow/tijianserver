**项目描述**：

- 本项目为两端架构：
- 用户端：完成体检预约全部业务流程。
- 医生端：完成对用户体检报告的编辑工作。
- 系统基于 Spring Boot 和 MyBatis 构建，利用 MySQL 进行数据存储，Redis 用于缓存。整个系统通过 RESTful API 进行前后端分离。

**项目亮点**：

- 使用springboot实现MVC架构
- 使用MyBatis实现基于ORM的数据层开发
- 使用Spring-AOP实现事务管理
- 高可用性：使用 Redis 进行数据缓存，大幅提高系统响应速度。
- RESTful API 设计: 使前后端分离更加清晰，更易于维护和拓展。
